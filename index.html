Политика конфиденциальности — Рецепты без интернета

1. Общие положения

Приложение работает полностью в офлайн-режиме и не собирает, не передаёт и не обрабатывает персональные данные пользователей.

2. Данные, используемые приложением

  • База данных, встроенная в приложение (копируется из ассетов на устройство при установке).
  • Архив изображений (ZIP), который загружается один раз с сервера разработчика и распаковывается на устройстве.

Эти данные не содержат персональной информации и не отправляются разработчику.

3. Разрешения приложения

Приложение может запрашивать доступ к памяти устройства только для сохранения и распаковки изображений. Никакие данные с устройства не передаются.

4. Хранение и безопасность

Все данные хранятся только на устройстве пользователя и могут быть удалены при удалении приложения.

5. Контактная информация

По вопросам, связанным с конфиденциальностью, свяжитесь с разработчиком: avk076@gmail.com

Права субъектов персональных данных

Если вы хотите запросить удаление, изменение или уточнение данных, свяжитесь с разработчиком по адресу электронной почты: avk076@gmail.com. Так как приложение не обрабатывает персональные данные, такие запросы выполняются немедленно.

Privacy Policy — Рецепты без интернета

1. General

This application works entirely offline and does not collect, store, or transmit any personal information.

2. Data Usage

  • The database included in the app package (copied from assets on installation).
  • Image archive (ZIP file) downloaded once from the developer's server and extracted locally.

No personal data is included or transmitted.

3. Permissions

The app may request access to device storage only to save and unpack image files. No data is sent outside the app.

4. Storage and Security

All app data is stored locally on the user's device and is removed when the app is uninstalled.

5. Contact

For privacy inquiries, contact: avk076@gmail.com

Data Subject Rights

If you want to request deletion, modification, or clarification of your data, contact the developer at avk076@gmail.com. Since the app does not process personal data, such requests are handled immediately.
